== Description ==



Wtyczka generuje pliki XML, umożliwiający integracje sklepu z serwisem ogłoszeniowym TwojaGielda.com



Połącz swój sklep internetowy z naszym serwisem i zyskuj podwójnie a wszystko to dzięki połączeniu poprzez plik XML.

Jakie korzyści daje Ci połączenie sklepu:

Pozyskanie dodatkowych odbiorców
Budowanie rozpoznawalności marki
Automatyczna aktualizacja listy produktów w naszym serwisie
Zwiększenie zysków



== Installation ==



= WYMAGANIA =



Aby korzystać z przywilejów jakie oferuje moduł niezbędne jest:



a)	Posiadanie konta na stronie TwojaGielda.com

b)	Zainstalowanie systemu sprzedaży Woocommerce:

c)	Pobranie plików instalacyjnych modułu z katalogu wtyczek Wordpress:






= INSTALACJA MODUŁU =



* Aktywuj wtyczkę
* Przejdź do strony "WooCommerce => Twoja Giełda" i zaktualizuj kategorię uruchamiania dla wszystkich witryn
* Na tej samej stronie zmapuj atrybuty odmian dla Twoja Giełda
* Przejdź do "Produkty => Kategorie". Dla każdej kategorii możesz:
    * wykluczyć kategorię z plików XML
    * zamapuj kategorie WooCommerce na kategorie witryn
* Przejdź do "Produkty". Dla każdego produktu możesz edytować ustawienia w zakładce "Twoja Giełda":
    * wykluczyć produkt z plików XML
    * Ustaw właściwości
* Pliki XML są generowane co trzy godziny. Aktualny status i adres URL do pliku XML można znaleźć na stronie ustawień "WooCommerce => Twoja Giełda"
* Musisz mieć zainstalowany i włączony moduł openssl na twoim serwerze, aby poprawnie zaktualizować kategorie. Może być również konieczne włączenie opcji "have allow_url_fopen" i "allow_url_include".
* Jeśli z jakiegoś powodu uaktualnienie kategorii nie działa, możesz je zaktualizować, wybierając ręczne przesyłanie dla każdej witryny. Pobierz odpowiedni plik, wybierz go z dysku twardego i uruchom aktualizację.
